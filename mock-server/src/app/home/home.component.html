Welcome to this small and useful Angular 6 application!


Through deeping into the code of this application, you will get some samples about how to use important elements of Angular 6.

Although some of these examples are basic, they offer a good entrypoint to develop others more complex.

Angular concepts:


  • Routing
  • Components
  • Reactive Forms
  • Reactive Forms Validators
  • Services
  • HTTP requests by RxJs
  • HTTP interceptors
  • HTTP Error Handling
  • Application Storage
  • Actions
  • Reducers
  • Effects
  • Angular Material (table)

Other stuff:


  • Docker
  • Docker Compose

This Angular application can also be started up by using Docker. Therefore, if needed, you can check that configuration as well. To achieve that, take a look at the files docker-compose.yml and Dockerfile.
